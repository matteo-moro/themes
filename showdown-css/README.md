CSS files in this folder are meant to be used with the Stylish extension.  
code1.css is to be set to apply to the play.pokemonshowdown.com domain.  
code2.css is to be set to apply to the replay.pokemonshowdown.com domain.  
code3.css is to be set to apply to the two domains above and to the psim.us domain.
